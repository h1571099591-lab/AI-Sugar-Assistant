# 🥗 AI 控糖助手

> 一款基于人工智能的糖尿病饮食管理工具，旨在帮助用户通过拍照快速识别食物热量，并获得个性化的饮食建议。

## ✨ 项目背景
本项目源于我在内分泌科的实习经历。我观察到许多糖尿病患者在出院后，由于缺乏专业、及时的指导，常常陷入“不知道怎么吃”的困境。因此，我设计并开发了这个应用，希望能将专业的医学知识，通过AI技术带给每一个需要的人。

## 🚀 功能特性
-   📸 **智能识别**：上传餐食图片，AI自动识别食物种类。
-   🔢 **热量估算**：基于识别结果，估算餐食总热量和营养构成。
-   💡 **控糖建议**：结合《中国2型糖尿病防治指南》，为用户提供通俗易懂的饮食建议。

## 🛠️ 技术栈
-   **前端/应用框架**：Streamlit
-   **AI 模型**：硅基流动 (SiliconFlow) API / Qwen2.5-VL-72B-Instruct
-   **核心语言**：Python 3.8+

## 📦 如何运行
1.  **克隆项目**：`git clone https://github.com/你的用户名/你的仓库名.git`
2.  **安装依赖**：`pip install -r requirements.txt`
3.  **配置密钥**：在项目根目录创建 `.env` 文件，并填入你的 `SILICONFLOW_API_KEY="你的API密钥"`
4.  **启动应用**：`streamlit run app.py`

## 🧑‍⚕️ 医学依据
本项目给出的饮食建议，主要参考《中国2型糖尿病防治指南》，旨在提供辅助性的健康管理参考，不能替代专业医生的诊断和治疗建议。
